"Aplikasi analisis konsumsi energi listrik interaktif dengan asisten cerdas Vecto. Dilengkapi fitur visualisasi vektor beban, simulasi penghematan real-time, dan deteksi dini kelebihan beban (anti-jepret)."
# ⚡ EnergiScan - Asisten Hemat Listrik

Aplikasi web interaktif untuk menghitung estimasi tagihan listrik bulanan berdasarkan golongan VA PLN, dilengkapi dengan asisten cerdas **Vecto**.

## ✨ Fitur Utama
- **AI Analisa (Vecto):** Memberikan saran penghematan berdasarkan input pengguna.
- **Visualisasi Vektor:** Menggambarkan beban listrik dalam bentuk vektor (Matematika).
- **Simulasi Hemat:** Menghitung potensi penghematan rupiah secara real-time.
- **Anti-Jepret Logic:** Peringatan otomatis jika beban melebihi kapasitas daya rumah.

## 🛠️ Tech Stack
- HTML5 & CSS3 (Glassmorphism UI)
- JavaScript (Vanilla)
- Chart.js (Data Visualization)
- Canvas API (Vector Graphics)
